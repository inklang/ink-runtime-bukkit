## Quick Start with Typeform

### Step 1: Create Account
1. Go to https://www.typeform.com
2. Click **"Sign up free"**
3. Create account with email or Google

### Step 2: Create Survey
1. Click **"Create typeform"**
2. Choose **"Start from scratch"**
3. Name it: **"Travyl App User Feedback Survey"**

### Step 3: Add Questions
Copy questions from `TRAVYL_SURVEY_FORM.md`:
- Use **"Rating"** question type for 1-5 scales
- Use **"Multiple choice"** for single selections
- Use **"Yes/No"** for boolean questions
- Use **"Long text"** for open feedback

### Step 4: Add Logic (Optional)
- Show Question 20 (bugs) only if they select bugs in Question 16
- Show Question 33-34 (support) only if they answer "Yes" to contacting support

### Step 5: Share
1. Click **"Share"** button
2. Copy the link
3. Share via:
   - Email campaigns
   - In-app notifications
   - Social media
   - Website popup

## Quick Start with Google Forms (Unlimited Responses)

If you need **unlimited responses** and don't mind basic design:

### Step 1: Create Form
1. Go to https://forms.google.com
2. Click **"+"** to create new form
3. Title: **"Travyl App User Feedback Survey"**

### Step 2: Add Questions
1. Click **"+"** to add question
2. Select question type:
   - **Linear scale** for 1-5 ratings
   - **Multiple choice** for single selection
   - **Checkboxes** for multiple selection
   - **Paragraph** for open text
   - **Short answer** for brief responses

### Step 3: Settings
- Enable **"Limit to 1 response"**
- Enable **"Collect email addresses"** (optional)
- Add **"Confirmation message"**

### Step 4: Share
1. Click **"Send"** button
2. Copy link or send via email
